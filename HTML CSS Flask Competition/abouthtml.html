About HTML

Hypertext Markup Language (HTML) is a language that is used to display documents in a web browser.

HTML is made up of several main components, including:

  • Tags, and Their Attributes
  • Data Types
  • Character References
  • Entity References
  • Elements

Tags and Their Attributes

Tags are what you write your HTML inside of. They consist of an opening tag and a closing tag.

An opening tag is shown by <and >, while a closing tag is shown by < and />

A common tag that you will see is the paragraph tag, which lets you write text. It is represented by <p>Your text here</p>

Some other common tags you will see include:

  • <h1> to <h6>, which are used to writing headings.
  • <br>, which is used to add a line break. Fun fact: It is one of few HTML tags that don't need to be closed!
  • Text modifying tags, such as <b> to bold text
