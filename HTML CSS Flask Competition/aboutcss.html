About CSS

Cascading Style Sheets, also known as CSS, is used to make your webpage look pretty.

You can have as many stylesheets as you need for an HTML file.

CSS can be used to define both the style and the layout of your webpage.

How do I use CSS?

To use CSS, you select an element, then you choose its property, and then you apply a value, as shown:

selector {
property: value;
}

An example:

p {
color: red;
}


This would change the color of the text in your paragraphs from black to red.


Each selector can have as many properties as you want, as shown:

body {
background: #FFFFFF;
Font-family: “Courier New”, Verdana, sans-serif;
}
